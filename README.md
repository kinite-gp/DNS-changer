# DNS-changer
 dns changer with python

 ## این برنامه به منظور سریع تر شدن فرایند تغییر dns نوشته شده


argument support :

   - auto
   - google
   - shecan
   - 1.1.1.1
   - electro
   - radar
   - 403
    
example :

   - python DNS.py -ip google
   - dns.exe -ip google
   - dns -ip google
   - python DNS.py --ip google
   - dns.exe --ip google
   - dns --ip google
